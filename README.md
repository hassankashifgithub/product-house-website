# Naissance — Born to Elevate

A premium, editorial e-commerce redesign concept for **Naissance US** (naissanceus.com) —
a woman-owned brand of handcrafted, limited-edition peplum belts made in Montclair, NJ.

Built as a single, self-contained `index.html` you can open directly in any browser.

## Highlights

- **Signature "Transform on Scroll" sequence** — a pinned, scroll-driven before→after
  that literally enacts the brand promise: *Transform the Pieces You Already Own.*
  (Inspired by the Vogue Gen-Z "outfit change on scroll" editorial.)
- **Immersive interactions** — custom cursor, film grain, scroll progress, kinetic
  hero typography, reveal-on-scroll, magnetic buttons, draggable horizontal lookbook,
  and a draggable before/after comparison slider.
- **Full editorial commerce** — The Edit (Statement $129 · Snap / Tie / Stretch $69 ·
  Minis $49–59), named Lookbook (Lumière Empress, Denim Queen, Crowned in Blue,
  Adinkra Grace, Golden Hour), Craft story, Philosophy, Heritage, and newsletter capture.
- **Real product photography** generated with the Higgsfield Soul 2.0 fashion-editorial
  model, served from CDN. Every image has a graceful labeled fallback so nothing
  ever renders empty.

## Brand
- Tagline: **Born to Elevate**
- Core message: **Transform the Pieces You Already Own**
- Voice: empowerment · confidence · transformation · femininity
- Woman-owned · handcrafted in the USA · Montclair, NJ

## Notes
- Open `index.html` in a browser (an internet connection loads the fonts + photography).
- To swap in the founder's real Instagram/product photos, replace the URLs in the
  `IMAGES` config object near the bottom of `index.html` — keys are self-describing.
